# Cyclistic User Behavior Analysis (2025)

## 📌 Project Overview
This project analyzes behavioral differences between annual members and casual riders to design a data-driven membership conversion strategy.

## 🎯 Business Objective
Annual members are more profitable than casual riders.  
The goal is to identify conversion opportunities and model potential revenue impact.

---

## 📊 Dataset

Source: Public historical trip data provided by Motivate International Inc.

Divvy Bike Share System (Chicago, USA)  
Access Link: https://divvy-tripdata.s3.amazonaws.com/index.html  

- Period analyzed: 12 months  
- Total records processed: 5.8M+  
- Data cleaned and transformed using SQL  

---

## 🛠 Technical Workflow

### 1. Data Cleaning (SQL)
- Removed invalid rides (<1 min or >24 hrs)
- Removed duplicates
- Handled null station data
- Created ride length, hour, day, and month features

### 2. Analysis
- Seasonal trends
- Weekday vs weekend behavior
- Peak hour patterns
- Electric bike preference
- Station traffic analysis

### 3. Visualization
Executive dashboard built in Tableau.

---

## 🔍 Key Insights

- Casual riders peak in August (13x January volume)
- 47% of casual rides occur in summer (June–August)
- Members show strong 5PM commuter spike
- Casual riders average longer trip duration
- Casual riders prefer electric bikes

---

## 💡 Strategic Recommendations

1. Behavior-based membership discounts  
2. Geo-targeted weekend campaigns  
3. Electric-bike membership bundles  

---

## 💰 Estimated Business Impact

Modeled Scenarios:

3% Conversion → $1.3M–$1.8M revenue increase  
5% Conversion → $2.3M–$3.0M revenue increase
